# Privacy Policy — Claude Usage Nyan（Claude 用量喵喵）

*Last updated: March 22, 2026*

---

## English

### What is Claude Usage Nyan?

Claude Usage Nyan is an open-source Chrome Extension that helps you track your Claude (claude.ai) usage quota and real-time token consumption. It runs entirely in your browser.

**Source code:** [https://github.com/zarqarwi/claude-usage-nyan](https://github.com/zarqarwi/claude-usage-nyan)

### Data Collection

**Claude Usage Nyan does not collect, transmit, or store any personal data.** Specifically:

- No analytics or tracking scripts
- No telemetry
- No user accounts or registration
- No data sent to any third-party server
- No advertisements

### Data Storage

All data generated by the extension (usage statistics, token counts, preferences) is stored locally on your device using `chrome.storage.local`. This data never leaves your browser.

### Network Requests

The extension makes network requests **only** to `claude.ai` — the official Anthropic website you are already using. These requests are made to:

1. **Retrieve your usage data** from Anthropic's usage API endpoints (the same data visible in your Claude Settings page)
2. **Intercept conversation API calls** on claude.ai pages to estimate real-time token consumption

No data is sent to any server other than `claude.ai`. No third-party APIs, services, or endpoints are contacted.

### Cookie Access

The extension reads your existing `claude.ai` session cookie solely to authenticate requests to the usage API. The extension:

- Does **not** store, copy, or cache your cookies
- Does **not** transmit your cookies to any third party
- Does **not** access cookies from any other website

### Permissions Explained

| Permission | Why it's needed |
|---|---|
| `storage` | Save usage data and preferences locally |
| `alarms` | Schedule periodic usage data refresh |
| `activeTab` | Display the floating status bar on claude.ai pages |
| `host_permissions: *://claude.ai/*` | Access Anthropic's usage API and intercept conversation data |

## 正體中文

### Claude 用量喵喵是什麼？

Claude 用量喵喵是一款開源 Chrome 擴充功能，幫助你追蹤 Claude（claude.ai）的用量配額與即時 token 消耗。它完全在你的瀏覽器中運行。

**原始碼：** [https://github.com/zarqarwi/claude-usage-nyan](https://github.com/zarqarwi/claude-usage-nyan)

### 資料收集

**Claude 用量喵喵不會收集、傳送或儲存任何個人資料。** 具體來說：

- 沒有分析或追蹤程式碼
- 沒有遙測（telemetry）
- 不需要帳號或註冊
- 不會向任何第三方伺服器傳送資料
- 沒有廣告

### 資料儲存

擴充功能產生的所有資料（用量統計、token 計數、偏好設定）都使用 `chrome.storage.local` 儲存在你的裝置本機。這些資料不會離開你的瀏覽器。

### 網路請求

擴充功能**僅**向 `claude.ai`（你正在使用的 Anthropic 官方網站）發出網路請求，用途為：

1. **取得你的用量資料**：呼叫 Anthropic 的用量 API（與你在 Claude 設定頁面看到的數字相同）
2. **攔截對話 API 呼叫**：在 claude.ai 頁面上即時估算 token 消耗

不會向 `claude.ai` 以外的任何伺服器傳送資料。不會聯繫任何第三方 API、服務或端點。

### Cookie 存取

擴充功能讀取你既有的 `claude.ai` session cookie，僅用於向用量 API 驗證身份。擴充功能：

- **不會**儲存、複製或快取你的 cookie
- **不會**將你的 cookie 傳送給任何第三方
- **不會**存取任何其他網站的 cookie

### 權限說明

| 權限 | 用途 |
|---|---|
| `storage` | 在本機儲存用量資料和偏好設定 |
| `alarms` | 排程定期更新用量資料 |
| `activeTab` | 在 claude.ai 頁面顯示浮動狀態條 |
| `host_permissions: *://claude.ai/*` | 存取 Anthropic 用量 API 並攔截對話資料 |
