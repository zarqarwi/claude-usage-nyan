# Privacy Policy — Claude Usage Nyan

Last updated: March 19, 2026

## Overview

Claude Usage Nyan is a Chrome extension for tracking your Claude usage on claude.ai. We respect your privacy.

## Data Collection

This extension does NOT collect, transmit, or store any personal data to external servers.

All data is stored locally in your browser (chrome.storage.local):
- Claude usage percentages (from claude.ai)
- Token estimation data
- Cost calculations

## Data Usage

- Uses your claude.ai session cookie to access Anthropic's usage API
- Cookies are ONLY used for requests to claude.ai, never sent to third parties
- All API requests go to claude.ai only

## Third-party Services

This extension uses NO third-party services: no analytics, no Firebase, no ads, no tracking.

## Permissions

- cookies: Read claude.ai session for API access
- storage: Store usage data locally
- alarms: Periodic data refresh (every 5 min)
- host_permissions (claude.ai): Access usage API endpoints

## Data Deletion

Uninstalling removes all local data. Use the Reset button to clear token tracking manually.
