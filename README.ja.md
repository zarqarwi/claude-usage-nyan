# 🐱 Claude 使用量にゃん — Usage Tracker

かわいい方法でClaudeの使用量を追跡しよう！

## 機能

- 📊 リアルタイムのセッション使用率＆週間使用量上限
- ⏰ リセットカウントダウンタイマー
- 🎨 かわいいグラデーションプログレスバー + シマーアニメーション
- 🟢🟡🟠🔴 一目でわかるカラーコード
- 🔄 5分ごとの自動更新 + 手動更新
- 🔒 すべてのデータはローカルに保存 — 外部サーバーへの送信なし

## インストール方法（デベロッパーモード）

1. Chromeを開き、`chrome://extensions/` にアクセス
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択
5. [claude.ai](https://claude.ai) にログインしていることを確認
6. ツールバーの 🐱 アイコンをクリックして使用量を確認

## 仕組み

この拡張機能は、claude.aiの既存のログインセッション（Cookie）を使用して、Claudeの内部使用量APIを定期的に呼び出し、使用量データを取得します。

**追加のAPIキーやセッションキーは不要です。**

## アーキテクチャ

```
manifest.json        — Chrome Extension MV3 設定
background.js        — Service Worker：データ取得 + バッジ更新
popup/
  popup.html         — ポップアップページ構造
  popup.css          — かわいいスタイルデザイン
  popup.js           — レンダリングロジック
content.js           — ブリッジレイヤー（isolated world）：claude.ai上のフローティングバー
injected.js          — メインワールドスクリプト：トークン追跡のためのAPIレスポンスインターセプト
icons/               — かわいい猫アイコン
```

## 注意事項

- Anthropicの内部APIには公開ドキュメントがなく、フォーマットはいつでも変更される可能性があります
- ポップアップに「生レスポンスデータ」が表示される場合は、APIフォーマットの解析ロジックの調整が必要です
- 初回使用時は、まずclaude.aiにログインしてから、拡張機能の更新ボタンをクリックしてください

## カスタマイズ

スタイルを変更したい場合は、`popup/popup.css` のCSS変数を編集してください：

```css
:root {
  --bg-cream: #FFF8F0;      /* 背景色 */
  --pink-accent: #FF7BAC;    /* メインアクセント */
  --mint-accent: #2DD4A0;    /* 安全色（緑） */
  /* ... その他の色はCSSを参照 */
}
```
